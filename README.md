# 城市更新背景下城市公园生态系统文化服务感知量化与更新优先级识别
# Quantifying Perceived Cultural Ecosystem Services and Identifying Renewal Priorities of Urban Parks in the Context of Urban Renewal

&gt; 基于大语言模型的城市公园公众感知智能识别与更新决策支持研究 | 以某市重点更新片区为例

## 摘要 Abstract

在城市发展由增量扩张转向存量提质的背景下，识别公众对公园生态系统文化服务（CES）的感知差异，是优化风景园林行动的重要依据。本研究以 **7 个重点更新片区公园**为对象，采集**新浪微博和小红书 2022 年 1 月至 2026 年 4 月公开评论 53,878 条**，运用**大语言模型多标签分类、情感分析和重要性—绩效分析（IPA）**方法，识别 **6 类生态系统文化服务**的感知特征及更新优先级。

**核心发现：**
- 审美价值和历史文化价值感知频率最高，游憩价值具有较强日常使用基础
- 教育价值和社会交往价值相对不足，存在较大提升空间
- 不同更新情境（历史文化保护 / 一江两岸风貌 / 新城商圈 / 老旧街区）下服务结构差异显著
- 基于 IPA 四象限提出分类施策的更新优先级建议

## 研究背景 Research Background

城市公园不仅承担生态调节和休闲游憩功能，也承载公共生活、地方记忆和文化体验等多重价值。传统研究多聚焦单个公园的设计策略，对**多公园、多类型、跨片区的系统评价和更新优先级判别**关注不足。

社交媒体文本分析、情感分析和大语言模型为公众感知的规模化识别提供了方法基础，但在城市更新重点片区公园评价中的应用仍有待深化。

## 主要结果 Key Findings

### 1. 感知频率与情感特征
- **审美价值**（56.8%）和**历史文化价值**（52.7%）感知频率最高
- **游憩价值**（35.0%）居中，日常使用基础扎实
- **精神价值**（10.5%）、**社会交往价值**（7.3%）、**教育价值**（5.2%）较低
- 情感得分整体处于**中性偏积极**区间

### 2. 不同更新情境的差异
| 更新情境 | 核心特征 |
|---------|---------|
| 历史文化保护与地标活化类 | 历史、景观、休闲并重 |
| 一江两岸风貌与线性空间更新类 | 审美价值最突出，滨水景观 + 慢行休闲 |
| 新城商圈与产城融合提质类 | 游憩与审美较高，历史文化偏低 |
| 老旧街区与枢纽片区整治类 | 审美价值突出，其余服务整体偏弱 |

### 3. IPA 更新优先级
- **优势保持区**：审美价值、游憩价值 → 持续维护景观品质
- **潜力激活区**：社会交往价值、精神价值 → 空间营造与活动组织
- **重点改进区**：历史文化价值 → 提升解说、互动和体验质量
- **常规观察区**：教育价值 → 嵌入游憩动线渐进培育
